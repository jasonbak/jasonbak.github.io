👨🏻‍💻 Professional info

I am a fourth year at Carnegie Mellon University studying statistics/machine learning and computer science as an undergraduate student. I am interested in machine learning, software engineering, and systems.

Last summer, I was at Amazon AWS on the Artificial Intelligence Algorithms team as a software developer engineer intern in Seattle, WA. I implemented features for the Random Cut Forest Cpp library with Python and Java bindings. The summer before that, I was at Fulcrum GT as a legal software developer intern in Chicago, IL, where I helped build a data cleansing and connection system.

Last year, I was a TA for CS 15-122, Principles of Imperative Programming (i.e., Data Structures), research assistant for the Institute of Software Research, and a multiple hackathon award winner.

Bringing ideas to life excites me. I’ve built a Messenger-based political assistant bot, compiled a comprehensive analysis of my personal journal, and coded a keyboard-based tab navigation Chrome extension. Currently, I’m working on an extensive class profile for the CMU SCS c/o 2019.
